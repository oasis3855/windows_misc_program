## Cursor32 マウス座標表示 for Windows

小さなウインドウにマウスカーソル座標を表示します。ユーザの指定した任意のキーを押すと、マウスの動きを垂直・水平に固定することができます

[ソフトウエアのダウンロードおよびドキュメントは cursor32](cursor32/README.md) (2004/04/17)
