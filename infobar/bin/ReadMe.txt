はじめに
      このプログラムは、作者が個人的に利用しているプログラムです。
      公開用に、あらゆる環境・条件でのテストを行っておりません。

      このプログラムでは以下のことが出来ます。

          ●電光掲示板風に、インターネットから取得したニュースを表示します。
          ●指定時間ごとに情報をアップデートできます。
          ●ユーザがインターネット接続条件（サイト、取得方法）をカスタマイズ
          できます。
          ●いくつかのニュースサイトをプリセットしています（約４０箇所）

１．動作環境
      Windows 95/98/Me/NT/2000/XP

      MFC42.DLL および MSVCRT.DLL がシステムフォルダに存在すること

２．使用法
      配布されたプログラム・パッケージを、適当なフォルダにコピーする。（新た
      にフォルダを作ることを薦めます。アンインストールに便利です。）

      初めて利用するときに、インストーラが起動します。

      プログラムが起動したら、ダイアログ上で右クリックして設定メニューを出す
      ことが出来ます。
      また、プログラムの終了も右クリックすることにより表示されるメニューから
      行います。
      ダイアログの位置を移動させるには、ダイアログ上で左クリックしたまま、ド
      ラッグしてください。
      ダイアログの隅をドラッグすると、サイズを変更できます。

      設定ダイアログの使い方の概要は、ヘルプファイルをご覧ください。
      なお、ありえない値や、プログラム上想定していないようなサイトにアクセス
      した場合の動作は不定です。

      情報を取得するホームページは、プリセットされたニュースサイトを想定して
      います。それ以外のホームページは作者が必要としていないため、テストして
      いません。
      想定されるホームページ以外での使用は、予期せぬエラーを引き起こすことが
      あります。
      （特に、最大文字数に注意してください。横8000ピクセル以上になる場合は、
      エラーを起こします）

      Windows ソケットは同期モードを使っています。（手抜き）
      応答が極端に遅いホームページや、転送が途中で止まってしまうようなホーム
      ページを指定しないでください。（スレッドが固まったまま投棄されることに
      なります）

      ★重要
      このプログラムは、たった３時間で作成した手抜きプログラムです。そのあた
      りを考慮の上、ご利用ください。（謎）
      （こんなしょぼいプログラムに、３時間もかけたのか！　と苦情を言うのもや
      めてください）


３．アンインストール方法
      スタートメニューの InfoBar アンインストール メニューを実行してください

４．使用許諾条件
      著作権は、井上博計が有します。
