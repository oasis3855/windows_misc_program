はじめに
      このプログラムでは以下のことが出来ます。
          ●バックライトを常時OFF／常時ON／システムによる自動制御に設定する
          ●ダイアログ版は動作確認のために利用
          ●トリガ版はホットキーに登録して使う形態を想定

１．動作環境
      Windows CE 4.2以降
      例：Pocket PC 2003, Windows Mobie 5.0, 6

      MFCCE300.DLL および COREDLL.DLL がシステムフォルダに存在すること

２．使用法
      Windows CEマシンにCABファイルをコピーし、ファイルエクスプローラなどで（
      ダブルクリックして）実行すると、セットアップが行われる。

      セットアップ時に「以前のバージョン用に設計されているため、正常に作動し
      ないか、またはまったく作動しない可能性があります。セットップを実行しま
      すか？」という警告が出たり、「このプログラムの発行者は不明です。続行し
      ますか？」という警告が出ますが、通常そのままインストールして問題無い。

      セットアップ後、プログラムメニューに「BacklightControl」と「BacklightC
      ontrol Trigger」の２つのショートカットが作成される。

      BacklightControl : ダイアログベースのプログラム
          ダイアログ上でバックライト制御（OFF・ON・自動）をテストできる

      BacklightControl Trigger : コマンドライン版 トリガプログラム
          バックライトをOFF・自動制御を切り替える


３．アンインストール方法
      WindowsCE設定コントロールパネルのプログラムの削除で「WindowsCE Backlig
      htControl」を削除する。
