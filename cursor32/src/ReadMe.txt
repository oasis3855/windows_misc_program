99/02/28 状況依存ヘルプファイルの追加
	hlp フォルダ以下のファイルを他のプロジェクトからコピーしてくる。
	ワークスペースに *.hpj,*.cnt と AfxDlg.rtfを加える
	*.hpj,*.cntにカスタムビルド情報をプロジェクトの設定から加える
		なお、このときデバッグ・リリース共通で加えること
	resource.hにもカスタムビルド情報を加えることで、状況依存型となる

	ヘルプファイルのジャンプIDは *.hpjでエリアスが宣言されているので
		*.hmファイルの全項目のエリアスを目次ページに割り当てる
	目次ページに#で始まる脚注を入れ、ジャンプ・ラベルとする
